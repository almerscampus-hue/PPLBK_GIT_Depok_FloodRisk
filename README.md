# PPLBK_GIT_Depok_FloodRisk
Projek "Pengembangan Aplikasi WebGIS untuk Analisis Kerawanan Banjir di Kota Depok" sebagai Tugas Akhir untuk Mata Kuliah Semester 6 "Proyek Perangkat Lunak Bidang Keilmuan Geospatial Information Technology" Teknik Informatika Universitas Ibn Khaldun Bogor

# PPLBK GIT - Flood Risk Depok

Project WebGIS analisis kerawanan banjir Kota Depok
menggunakan Python, Google Earth Engine, dan QGIS.

## Parameter
- Slope
- Rainfall
- River Distance
- Land Cover

## Tools
- Google Colab
- Python
- QGIS
- Flask
- Leaflet
